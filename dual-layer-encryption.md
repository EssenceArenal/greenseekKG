# Dual-Layer Encryption

> **Definition:** Dual-layer encryption lets a GreenSeek user set a personal encryption password on any project or chat thread. Once enabled, the content is locked behind the user's own access code and is inaccessible to anyone — including GreenSeek itself.

## Frequently asked questions

**What happens if I lose my encryption password?**

Because GreenSeek cannot access dual-layer-encrypted content, a lost password means the content cannot be recovered. That is the guarantee working as designed.

**Is dual-layer encryption available on all plans?**
