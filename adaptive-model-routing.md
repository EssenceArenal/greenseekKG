# Adaptive Model Routing

> **Definition:** Adaptive model routing automatically selects the right-sized AI model for each task: small, efficient models handle everyday work, while deeper models activate only when a task needs them. GreenSeek uses adaptive routing to cut energy use and cost without sacrificing quality on hard problems.

## Frequently asked questions

**Does adaptive routing reduce answer quality?**

No — routing is about matching task difficulty to model capability. Complex tasks still reach the deepest models.

**Why does routing matter for sustainability?**

Most AI energy is spent running oversized models on simple tasks. Right-sizing each request is the single biggest efficiency lever.
