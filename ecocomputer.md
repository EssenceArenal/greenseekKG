# EcoComputer & Platform Features

EcoComputer is GreenSeek's workspace concept: Research, Create, and Automate in one place. Supporting features from the product knowledge base:

- **Chat & AI Modes** — response modes Eco, Standard, Deep, Ultra; file uploads; encrypted chat mode; name redaction mode; web search; conversation export; temporary (incognito) chats.
- **Custom Experts** — user-created AI experts with domain rules, reasoning frameworks, uploaded knowledge files, "Optimize Expert Knowledge", guardrails and disclaimers.
- **Projects** — AI workspaces with document-grounded chat (RAG), file uploads, sharing (including to non-account holders), project tasks and channels, and conversion of documents into structured knowledge.
- **Teams** — team workspaces, member invites, roles, shared projects.
- **Cloud Tasks** — automated AI tasks with tools, schedules, and privacy-mode support.
- **Workflows** — guided multi-step research and document outputs that can use project knowledge.
- **Deck Builder / Deck Studio** — AI-generated presentations, themes, and analysis of uploaded decks.
- **Image Generation & Vision** — generate images; send images for analysis.
- **File Vault** — central file storage, processing, YouTube transcript import.
- **API** — OpenAI-compatible API with keys, streaming, and rate limits.
