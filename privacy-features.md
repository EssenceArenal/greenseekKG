# Privacy Features — Private by Design

GreenSeek's position: ethical intelligence includes protecting your data, with tools for sovereign AI experiences. Encryption modes, name redaction, project isolation, and guaranteed data privacy.

## Redaction Modes
Intelligent redaction configured in Settings: HIPAA-compliant modes for healthcare workflows, personal privacy presets that strip names and identifiers, and custom rules per industry.

## Dual-Layer Encryption
A personal encryption password on any project or chat thread. With dual-layer encryption enabled, conversations are locked behind the user's own access code — inaccessible to anyone, including GreenSeek.

## Incognito Mode
Browse, research, and query without leaving a trace. Disables session history, prevents logging, and ensures nothing from the session is retained or used to improve models.

## Data & sovereignty
The public knowledge base covers: where data is stored, whether data trains models, available encryption, name/detail redaction, full data export and deletion, and sovereign AI. Always quote these docs exactly — never paraphrase privacy guarantees.
